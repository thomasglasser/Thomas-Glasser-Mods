Welcome to Thomas Glasser Mods! This is a website where I post my mods for the Minecraft: Java Edition game. I also take commissions for mods, so if you want a mod made, feel free to contact me!

I currently have one released mod, Sherds API, and 1 mod in development, Minejago. Updates and support are offered on my Discord server.

Latest Mod: Sherds API

Sherds API provides a simple way for mods and datapacks to add new sherd patterns to the game with two json files and a texture.
